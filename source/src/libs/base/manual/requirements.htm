Systemvoraussetzungen

Die Anforderungen für Beeco sind relativ simpel.
Es wird ein wenig WebSpace benötigt mit mind. PHP4 und den Image Funktionen (GD Library mit FreeType [TTF] Support). Dazu eine mySQL Datenbank mit ausreichenden Rechten (INSERT, SELECT, UPDATE, DELETE, CREATE, DROP). Des weiteren einen temporären Ordner mit Schreibrechten für den Webserver (chmod 777). Standardmäßig wird der von PHP vorgegeben upload Ordner verwendet. Dies kann in der Konfigurationsdatei other.settings.php in /user/settings/base/ jedoch angepasst werden. Sollten Sie das Plugin "Bildergalerie" (PICS) benutzen, wird ein seperater Upload Ordner im user Verzeichnis verwendet. Der ebenfalls ausreichend Schreibrechte haben muss. Der Pfad und Name des Ordner kann in der Konfigurationsdatei /user/settings/plugins/pics/ geändert werden.

Die Administrations Oberfläche benötigt lediglich einen JavaScript fähigen Browser, indem dies natürlich auch aktiviert sein muss. Sofern Sie den Internet Explorer benutzen und den kostenlosen Editor (FCKEDITOR) sollten Sie Ihren Browser so konfigurieren, dass dieser Cookies akzeptiert. In den meisten Fällen ist dies bereits aktiviert.
