Installation

Für die Installation von Beeco wird auf dem Server (WebHoster) PHP4 mit MySQL Zugriff benötigt.

1. Zuerst muss das DUMMY Paket in den Zielordner kopiert werden in dem die neue Beeco Seite
benutzt werden soll.

2. Danach müssen die Quellen (Sources) ebenfalls dort hinein kopiert werden, indem der Ordner "src"
einfach in dem DUMMY Paket Ordner kopiert wird. Der SRC Ordner kann auch einfach nur verlinkt werden
(Symbolic Links)

3. Dann müssen Sie MySQL Verbindungsdaten in die Konfigurationsdatei eintragen.
Gehen Sie dazu in das folgende Verzeichnis im Beeco Ordner: "user/settings/base/"

4. Anschließend öffnen Sie die Datei "mysql.settings.php" und tragen dort an beschriebener Stelle
die MySQL Verbindungsdaten ein.

Der Table_Prefix steht für die Datenbank Tabellen, der neuen Beeco Installation.
Durch unterschiedliche Table_Prefix Bezeichnungen, ist es möglich
mehrere Beeco Installationen innerhalb einer Datenbank laufen zu lassen.

5. Wenn Sie Ihre Daten eingetragen haben, speichern Sie die Datei ab und öffnen im Browser
die Internetseite die zu dem Beeco führt. Hinter dieser Adresse fügen Sie noch ein /install an.
Bsp.: http://www.ihredomain.de/cms/install

6. Füllen Sie die dort vorgegeben Felder aus. Bei Fragen klicken Sie auf das Fragezeichen (?).
