Hinzufügen

1. Allgemeines

Über den Menüpunkt "Hinzufügen" erstellen Sie neue Menüpunkte für die Beeco verwaltete Webseite. Es gibt keine Begrenzung an Menüpunkten und untergeordneten Menüpunkten. Um die untergeordneten Menüpunkte eines Menüpunktes zu sehen, klicken Sie auf den Titel des Menüpunktes.

Einen neuen Menüpunkte erstellen Sie, indem Sie ein Titel (der Name des neuen Menüpunktes) eingeben, einen Typ auswählen und anschließend auf "Hinzufügen" klicken. Der Menüpunkt wird automatisch an die Position gesetzt an der Sie sich befinden. Wenn Sie also zuvor auf den Titel eines bestehenden Menüpunktes geklickt haben, wird der neue Menüpunkt diesem Menüpunkt untergeordnet.

 

2. Typen

Es gibt drei Typen von Menüpunkten: "Inhalt", "Weiterleitung", "Manuell" und "Plugin".
Wenn Sie Typ "Inhalt" ausgewählt haben, wird ein Menüpunkt erstellt mit Ihrem individuellen Text, den Sie in das darauffolgende Formular eingeben können.

Die andere Möglichkeit ist, Sie erstellen einen Menüpunkt als Typ "Weiterleitung". Hierbei wird ein Menüpunkt erstellt der den Nutzer zu einer völlig anderen Seite leitet (wie z.B. Beeco unabhängige Seiten). Als URL geben Sie einfach in das darauffolgende Formular die Zieladresse ein. Wichtig hierbei ist, wenn Sie auf eine fremde Internetseite (www.fremdedomain.de) weiterleiten möchten, müssen Sie zu beginn http:// schreiben und dann die fremde Internetseite eingeben. Somit erkennt der Browser des Besuchers, dass die angegebene Adresse nicht auf Ihrer Internetseite zu finden ist.
Bsp: http://www.fremdedomain.de

Des weiteren haben Sie noch die Möglichkeit einen "manuellen URL" zu erstellen. Das Prinzip ist ähnlich wie bei der Weiterleitung. Allerdings wird der Menüpunkt im klartext erstellt. Das heißt dass der User nicht durch das Beeco zu einer anderen Seite weitergeleitet wird, sondern Sie selber entscheiden was der Link für ein Ziel enthält, bzw. welche Funktion erfüllt wird. Somit ist es z.B. möglich javascript direkt in den Menüpunkt einzubinden. Ein Beispiel für ein manuellen URL könnte zum Beispiel folgender Inhalt sein: javascript:alert('beispiel');
Wenn mit Hilfe dieses manuellen URLs der Besucher Ihrer Internetseite nun auf den manuellen URL klickt, erscheint eine kleine Warnmeldung mit dem Text "beispiel". Der User wird aber nirgends weitergeleitet und es wird kein weiterer Inhalt angezeigt.

Als letzte Menüpunkt Art gibt es noch die "Plugins" hierbei haben Sie auf einfachste Weise die Möglichkeit, einen Menüpunkt anzulegen der auf ein installiertes Plugin verweist. Sie müssen lediglich auswählen, auf welches Plugin der Menüpunkt verweisen soll.

 

3. Priorität

Alle Menüpunkte werden grundsätzlich nach dem Alphabet sortiert. Dies kann aber sicherlich etwas störend sein, wenn man die Menüpunkte anders sortiert haben möchte. Hierbei ist die Priorität behilflich. Wenn Sie z.B. Priorität 0 für den Menüpunkt "Home" eingeben und Priorität 1 für den Menüpunkt "Anfahrt". Hat der Menüpunkt "Home" vorrang vor "Anfahrt". Somit lässt sich die Sortierung beeinflussen. Menüpunkte mit gleicher Priorität werden wieder nach Alphabet sortiert.
