DirectADD

1. Allgemeines

DirectADD steht für direktes Hinzufügen. Mit dieser Funktion können Sie Teile der verfügbaren Plugins, direkt in Ihrem Text eines Menüpunktes nutzen. Dadurch können Sie zum Beispiel einen einzelnen Gästebuch Eintrag von dem Gästebuch Plugin direkt in Ihrem Text einzufügen. Oder Sie füge ein Bild aus dem Galerie Plugin direkt in Ihrem Text ein ohne auf die Galerie verweisen zu müssen.

 

2. Nutzen von DirectADD

Jedes Plugin hat einen Namen und einen Kurznamen (z.B. Gästebuch bzw. gbook). Diese Namen werden in der roten Titelleiste angezeigt. Unter der roten Leiste finden Sie in Klammern den Kurznamen (z.B. gbook). Dieser Kurzname wird benötigt! Wenn Sie einen neuen Menüpunkt (Typ: Inhalt) hinzufügen oder bearbeiten, nutzen Sie die DirectADD Funktion wie folgt: {kurzname:optionen}

Bsp: {gbook:last}
Die genauen Optionen, sind bei jedem Plugin anders. Im nächsten Absatz finden Sie die Optionen für das aktuelle Plugin in dem Sie sich gerade befinden.

 

3. Optionen

Für die News stehen Ihnen folgende Optionen zur Verfügung:

last = Zeigt immer den zuletzt eingetragenen News Eintrag an. Bsp: {news:last}
NR = Anstatt NR verwenden Sie die Nummer die vor einem News Eintrag steht um diesen anzuzeigen. Bsp: {news:2}
last-NR = Zeigt den letzten NR Eintrag an. Bsp: {news:last-2} Zeigt den vorletzten Eintrag an.
