Als Bild geben Sie eine große Version des Bildes an (kein Vorschau Bild). Es wird automatisch auf die passenden Größen umgerechnet wie es in den Einstellungen definiert wurde. Achten Sie darauf dass die Dateigröße des Bildes nicht der unten angegebenen Dateigröße überschreitet. Es werden nur JPEG (JPG) Bilder akzeptiert!

Klicken Sie anschließend nur einmal auf "Hinzufügen". Je nach Internetverbindung kann das Hinzufügen der Bilder einige Minuten dauern!
