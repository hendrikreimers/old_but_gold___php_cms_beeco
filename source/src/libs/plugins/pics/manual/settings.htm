Einstellungen

1. Allgemeines

In den Einstellungen können Sie die Darstellung der Bildergalerie veränden. Zum Beispiel wie viele Bilder in einer Reihe und auf einer Seite angezeigt werden.

 

2. Einträge pro Zeile

Die Bilder werden bei jeder Gruppe in einer kleinen Übersicht angezeigt bevor das große Bild angezeigt wird. Bei der Übersicht werden die Bilder, aber nicht alle nebeneinander angezeigt, sondern ab einer gewissen Menge wird eine neue Zeile (Absatz) angefangen und die nächsten Bilder angezeigt. Mit dieser Einstellung geben Sie an, wieviele Bilder maximal in jeder Zeile stehen dürfen.

 

2. Anzahl der Zeilen

Damit es nicht zu viele Zeilen auf einmal gibt und somit die Galerie schnell unübersichtlich, kann man auch die Anzahl der Zeilen begrenzen. Wenn zum Beispiel 4 Zeilen erreicht sind, es aber noch eine fünfte benötigt wird. Werden alle weiteren Bilder auf einer neuen Seite verschoben, durch die man mit "VOR" und "ZURÜCK" blättern kann.

 

3. Darstellung

Wenn Sie auf ein Bild klicken erscheint es größer und darunter steht der Titel sowie Ihre Beschreibung. Das geschieht alles auf Ihrer Beeco Internetseite. Je nach größe der Detailbilder kann es aber auch stören wenn alles dort erscheint. Aus diesem Grund können Sie die Darstellung der Detailansicht ändern. Wenn Sie das "Fenster (PopUp)" aktivieren erscheint ein völlig neues Fenster, dass sich der größe des Bildes automatisch anpasst. Durch einblenden verdunkelt sich die Webseite und das Bild erscheint im selben Fenster auf der Webseite. Diese beiden Optionen benötigen allerdings, dass im Browser JavaScript aktiviert ist. Dies ist allerdings in den meisten Fällen aktiviert. Die letzte Darstellungsmethode ist "Standard". Bei dieser erscheint das Bild wie oben erwähnt, innerhalb der Internetseite und legt sich nicht auf die abgedunkelte Webseite darauf.

 

4. Sortierung

Durch die Sortierung Abwärts/Aufwärts werden die Bildergruppe bzw. Bilder in einer anderen Reihenfolge dargestellt. Als Beispiel: Aufwärts sortierung ist als ob Sie von A-Z zählen würden. Aufwärts sortierung ist von Z-A.

 

5. Automatische Bildanpassung

Die Bilder zu verkleinern und für das Internet optimal zu speichern ist aufwendig und schwierig. Zudem benötigt man häufig kostenpflichtige Programme die einem die Bilder auf die gewünschte Größe reduzieren um diese in die Bildergalerie einzufügen. Wenn Sie sich diese Arbeit ersparen möchten, aktivieren Sie "Automatische Bildanpassung". Sie geben Ihr Bild von Ihrem Computer an und Beeco wandelt es automatisch für die Bildergalerie um, so dass es optimal hineinpasst.

Die Größe der Bilder die abgespeichert werden, können Sie auch hier beeinflussen. Geben Sie die gewünschte maximale Bildbreite (in Pixel) dazu ein. Beeco errechnet an der von Ihnen angegeben Breite ein optimales Verhältnis zur Höhe und speichert dann das Bild ab. Die Höhe kann nicht angegeben werden, da nicht jedes Bild genau die gleiche Größe hat. Diesen Schritt übernimmt Beeco. Es erkennt Bilder zum Beispiel die im Hochformat erstellt worden sind und nimmt die von Ihnen angegebene Breite einfach als Höhe und errechnet dann die passende Bildgröße. Somit sind alle Bilder gleich groß.

Hinweis: Wenn ein Bild, dass Sie hochladen kleiner als die angegebene bzw. ausgerechnete Bildgröße ist, bleibt das Bild unverändert. Lediglich das Vorschaubild wird angepasst.
