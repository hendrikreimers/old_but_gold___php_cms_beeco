DirectADD

1. Allgemeines

DirectADD steht für direktes Hinzufügen. Mit dieser Funktion können Sie Teile der verfügbaren Plugins, direkt in Ihrem Text eines Menüpunktes nutzen. Dadurch können Sie zum Beispiel einen einzelnen Gästebuch Eintrag von dem Gästebuch Plugin direkt in Ihrem Text einzufügen. Oder Sie füge ein Bild aus dem Galerie Plugin direkt in Ihrem Text ein ohne auf die Galerie verweisen zu müssen.

 

2. Nutzen von DirectADD

Jedes Plugin hat einen Namen und einen Kurznamen (z.B. Gästebuch bzw. gbook). Diese Namen werden in der roten Titelleiste angezeigt. Unter der roten Leiste finden Sie in Klammern den Kurznamen (z.B. gbook). Dieser Kurzname wird benötigt! Wenn Sie einen neuen Menüpunkt (Typ: Inhalt) hinzufügen oder bearbeiten, nutzen Sie die DirectADD Funktion wie folgt: {kurzname:optionen}

Bsp: {gbook:last}
Die genauen Optionen, sind bei jedem Plugin anders. Im nächsten Absatz finden Sie die Optionen für das aktuelle Plugin in dem Sie sich gerade befinden.

 

3. Optionen

Für eine der DirectADD Funktionen benötigen Sie eine ID für ein Bild. Die ID von Bildern wird ihnen beim bearbeiten oder löschen von Bildern angezeigt. Alle Optionen bestehen aus 2 Teilen. Einmal ob das Bild groß oder klein angezeigt werden soll (small oder big) und dann der eigentlichen Option mit Komma getrennt (siehe unten).

last = Zeigt immer das zuletzt eingefügt Bild an (sofern die Gruppe aktiv ist): {pics:small,last} oder {pics:big,last}
ID = Anstatt ID verwenden Sie die Nummer die vor einem Bild um das anzuzeigen. Bsp: {pics:small,2} oder {pics:big,2}
rand = Zeigt ein Bild aus allen Gruppen zufällig an Ändert sich bei jedem neuen Aufruf. Bsp: {pics:small,rand} oder {pics:big,rand}
short = Wenn Sie ein Bild einfügen möchten, es soll aber nicht auf die Gallerie verweisen (Link), können Sie zu den oberen 3 Optionen noch ",short" anhängen. Bsp: {pics:small,last,short} oder anderes Beispiel {pics:big,rand,short}

Hinweis: Sollten Sie die Darstellungsmethode "Einblenden" in den Einstellungen aktiviert haben, müssen Sie den DirectADD Befehl {pics:ajax} zu beginn des Inhalts (und somit vor beginn der anderen DirectADD Funktionen einmalig eingefügt haben. Dadurch wird gewährleistet, dass die für das Einblenden benötigten Funktionen eingebunden werden.

Beispiel:
   Richtig: {pics:ajax} Der Text {pics:small,21} noch mehr Text {pics:small,rand} viel mehr Text.
   Falsch: Der Text {pics:small,21} noch mehr Text {pics:small,rand} viel mehr Text. {pics:ajax}
