Hinzufügen

1. Allgemeines

Wie mehrere Fotoalben werden auch hier Bilder einem "Fotoalbum" (Gruppe) zugeordnet. Um eine Gruppe zu erstellen geben Sie in dem oberen Eingabefeld einen Titel und gegebenfalls eine kleine Kurzbeschreibung für die neue Gruppe ein. Anschließend klicken Sie auf hinzufügen. Klicken Sie direkt auf den Titel einer Gruppe um dort ein neues Bild hinzufügen zu können. Es können nur JPG (JPEG) Bilder hochgeladen werden.

Jedes Bild kann einen Titel und eine Beschreibung haben, die Sie dort eingeben und später auch bearbeiten können. Wenn Sie in den Einstellungen "Automatische Bildanpassung" aktiviert haben, brauchen Sie nur Ihr Bild von Ihrem Computer auswählen. Es wird automatisch in das für die Galerie passende Format umgewandelt. Das hochladen von Bildern von Ihrem Computer in die Bildergalerie kann je nach Bildgröße und Internetverbindung etwas dauern. Warten Sie daher bis eine Bestätigung erscheint, dass das Bild zur Gruppe hinzugefügt wurde.

Sollten Sie die "Automatische Bildanpassung" nicht aktiviert haben, ist es notwendig dass Sie 2 Bilder angeben. In dem ersten Feld das kleine Vorschau Bild (für die Übersicht der Bilder) und ein Detailbild (das selbe Bild nur viel Größer zur genaueren Betrachtung). Hierbei gilt auch, dass das hochladen von Bildern etwas dauern kann. Warten Sie daher bist eine Bestätigung erscheint, dass das Bild hinzugefügt wurde.
